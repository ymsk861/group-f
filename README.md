## 授業中に生徒の表情を認識し理解度をフィードバックするアプリケーション
[Hait Lab](https://hait-lab.com/) Primary Course 4th Fチームの制作物  

## Environment
Mac OS / ubuntu18.04  
Python3

## Installation
- Python Packages
```
$ pip install -r requirements.txt
```
- AWS CLI
```
$ aws configure
```

## Usage
```
$ python detect_faces.py
```
